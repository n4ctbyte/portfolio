Format Gambar

Berikut adalah 10 format gambar populer yang umum digunakan dalam desain dan pengembangan web.

×

.JPEG / .JPG

Joint Photographic Experts Group

Format kompresi lossy yang ideal untuk foto dan gambar dengan banyak warna. Tidak mendukung transparansi.

Kelebihan: Ukuran file kecil, kompatibilitas tinggi

Kekurangan: Tidak mendukung transparansi, kualitas berkurang dengan kompresi

.PNG

Portable Network Graphics

Format kompresi lossless yang mendukung transparansi, ideal untuk logo, ikon, dan grafik.

Kelebihan: Mendukung transparansi, kualitas tinggi

Kekurangan: Ukuran file lebih besar dari JPEG

.GIF

Graphics Interchange Format

Format yang mendukung animasi sederhana dan transparansi. Terbatas pada 256 warna.

Kelebihan: Mendukung animasi, transparansi

Kekurangan: Terbatas pada 256 warna

.ICO

Icon File Format

Format khusus untuk ikon yang digunakan oleh Windows. Mendukung berbagai ukuran dan transparansi dalam satu file.

Kelebihan: Dapat menyimpan beberapa ukuran ikon dalam satu file, dikenal sebagai favicon

Kekurangan: Dukungan terbatas pada web, fungsi khusus untuk ikon

.BMP

Bitmap Image File

Format tidak terkompresi yang menyimpan data gambar piksel per piksel. Digunakan oleh Windows.

Kelebihan: Tidak ada kehilangan kualitas

Kekurangan: Ukuran file sangat besar

.SVG

Scalable Vector Graphics

Format berbasis vektor yang dapat diubah ukurannya tanpa hilang kualitas, ideal untuk logo dan ikon.

Kelebihan: Skalabel tanpa hilang kualitas, dapat dianimasi

Kekurangan: Tidak ideal untuk foto kompleks

.RAW

Raw Image Format

Format mentah dari kamera digital yang menyimpan semua data yang ditangkap oleh sensor kamera.

Kelebihan: Fleksibilitas penyuntingan maksimal, kualitas tertinggi

Kekurangan: Ukuran file sangat besar, memerlukan pemrosesan

.AVIF

AV1 Image File Format

Format modern dengan kompresi tinggi yang menawarkan kualitas lebih baik dari JPEG dan WebP dengan ukuran file lebih kecil.

Kelebihan: Kompresi sangat efisien, kualitas tinggi, mendukung transparansi

Kekurangan: Dukungan browser masih berkembang, pemrosesan lebih intensif

.EPS

Format EPS tidak didukung oleh browser

Encapsulated PostScript

Format vektor yang banyak digunakan dalam desain grafis profesional dan percetakan, mendukung grafik dan teks.

Kelebihan: Skalabel, kompatibel dengan software desain profesional

Kekurangan: Kompleksitas tinggi, dukungan web terbatas

.WebP

Web Picture Format

Format modern yang menawarkan kompresi lebih baik dari JPEG dan PNG, mendukung transparansi dan animasi.

Kelebihan: Ukuran file kecil dengan kualitas tinggi, mendukung transparansi

Kekurangan: Kompatibilitas terbatas dengan browser lama
